## 使用说明
### 安装依赖
```
pip3 install -r requirements.txt
```
### 使用脚本
```
cd scripts
# 预训练模型，从图片识别获取车牌
python pre_train.py

# 对单张图预处理
python detect.py

# 获取车牌
python check.py
```
